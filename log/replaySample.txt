# Everything before the first `=== ...` separator is discarded by the parser,
# so this preamble can hold free-form notes (the parser has no comment syntax —
# `#` is just decoration).
# A few editing shortcuts that aren't worth putting in the manual:
#   - The `=== ...` header is a free-form label shown in the replay-selection menu.
#     Writer emits a timestamp, but you can replace it with any name (e.g.
#     `=== replaySample`) — the text after `=== ` becomes the menu entry.
#   - The bidding section (the ` a   b   c   d   e ` header and `S12  -  C13 ...` rows
#     before the first `---`) is optional. If omitted, replay assumes Napoleon declared
#     the recorded bid once and everyone else passed.
#   - The cumulative honor counts to the right of `|` on each trick line are optional.
#     Drop the `|` and everything after it and the trick still parses.
#   - The final score-delta line (e.g. ` +6   -4   -4   -4   +6`) is optional.
